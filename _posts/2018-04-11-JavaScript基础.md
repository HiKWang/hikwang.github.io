# 变量

## 顶置

>> 如果你写一个声明和初始化变量的多行JavaScript代码的程序，你可以在初始化变量之后再实际声明它，并且它仍然可以工作。这是因为变量的声明通常在在其余的代码执行之前完成。这叫做顶置。

```
bla = 2;
var bla;
// ...

// is implicitly understood as:

var bla;
bla = 2;
```
